Como a√±adir elementos a un gr√°fico

f <- function(x){
  x^2-2*x+sqrt(abs(x))
}
plot(f, xlim = c(-3,3)) # graficamos la funci√≥n

abline(h = 5, lty = "dotdash",lwd = 3, col = "gray") # Linea Horizontal
